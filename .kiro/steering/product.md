# Product Overview

## Zella Trade Scribe

A comprehensive trading journal application designed for forex and futures traders. The platform provides professional-grade trade tracking, analytics, and journaling capabilities similar to Tradezella, Tradervue, and TraderSync.

### Core Features

- **Trade Management**: Complete CRUD operations for trades with real-time Firebase synchronization
- **Multi-Account Support**: Track multiple trading accounts (live/demo) across different brokers and platforms
- **Advanced Analytics**: Performance metrics, win rates, profit factors, R-multiples, and visual charts
- **Daily Journaling**: Calendar-based journaling with templates, trade integration, and reflection tools
- **Strategy Tracking**: Trading playbooks, setup classification, and pattern recognition
- **Risk Management**: Position sizing, partial closes, stop-loss tracking, and risk analytics

### Target Users

Professional and semi-professional forex/futures traders who need detailed trade tracking, performance analysis, and systematic journaling for trading improvement.

### Key Differentiators

- Firebase-powered real-time synchronization across devices
- Comprehensive forex-specific features (pips, lots, currency pairs, sessions)
- Advanced setup classification and pattern recognition system
- Integrated daily journaling with trade correlation
- Mobile-responsive design with accessibility features
