# Daily Journal User Guide

## Getting Started

The Daily Journal redesign provides a powerful, calendar-based approach to trading reflection and market analysis. This guide will help you make the most of its features.

## Accessing the Daily Journal

1. **From the sidebar**: Click "Daily Journal" in the main navigation
2. **From TradeLog**: Click "View Notes" on any trade with journal entries
3. **Direct URL**: Navigate to `/daily-journal` or `/daily-journal/2024-01-15`

## Navigation

### Week Navigator
- **Previous/Next Week**: Use arrow buttons to navigate between weeks
- **Current Week**: Click the center button to jump to the current week
- **Quick Date Selection**: Click the date picker icon to select any specific date

### Calendar View
The week-based calendar shows:
- **Daily P&L**: Green for profitable days, red for losses
- **Trade Count**: Number of trades executed each day
- **Journal Status**: Indicators showing completion status
- **Click any date** to navigate directly to that day

## Content Modes

### Daily Journal Mode
Perfect for:
- Market analysis and observations
- Trading psychology reflection
- Daily performance review
- General trading thoughts

### Trade Notes Mode
Ideal for:
- Specific trade analysis
- Screenshot documentation
- Trade-linked reflections
- Performance attribution

**Switching Modes**: Use the tabs at the top of the content area. Content is preserved when switching between modes.

## Using Templates

### Applying Templates
1. Click the "Templates" button
2. Browse available templates
3. Select your desired template
4. Click "Apply Template"
5. Customize the content as needed

### Creating Custom Templates
1. Open the template selector
2. Click "Create Template"
3. Enter template name and content
4. Use placeholders like `{date}`, `{pnl}`, `{trades}`
5. Save for future use

### Template Examples

## Trade Integration

### Linking Trades to Notes
1. Switch to "Trade Notes" mode
2. Select a trade from the dropdown
3. Trade data automatically displays (P&L, prices, timing)
4. Add your analysis and reflections
5. Upload screenshots if needed

### Screenshot Management
- **Drag & Drop**: Simply drag image files onto the upload area
- **File Browser**: Click "Upload Screenshot" to browse files
- **Size Limit**: Maximum 5MB per image
- **Supported Formats**: PNG, JPG, JPEG, GIF

### Viewing Trade Details
When a trade is selected, you'll see:
- Currency pair and direction
- Entry and exit prices
- Position size and P&L
- Trade timing and duration
- Strategy and setup information

## News Events

### Adding Market Events
1. Click "News Events" button
2. Click "Add Event"
3. Fill in event details:
   - Title (e.g., "ECB Interest Rate Decision")
   - Time (e.g., "14:00")
   - Impact level (High/Medium/Low)
4. Save the event

### Managing Events
- **Edit**: Click the edit icon next to any event
- **Delete**: Click the delete icon and confirm
- **Filter**: Events are organized by date and impact level

## Troubleshooting

### Common Issues

**Can't see my trades**
- Check that you have the correct account selected
- Verify the date range includes your trades
- Refresh the page if data seems stale

**Templates not loading**
- Clear your browser cache
- Check your internet connection
- Try refreshing the page

**Screenshots won't upload**
- Check file size (must be under 5MB)
- Verify file format (PNG, JPG, JPEG, GIF)
- Try a different browser if issues persist

**Navigation feels slow**
- Close other browser tabs to free memory
- Check your internet connection speed
- Clear browser cache and cookies

## Advanced Features

### URL Parameters
You can bookmark specific dates and trades:
- `/daily-journal/2024-01-15` - Go to January 15, 2024
- `/daily-journal/2024-01-15/trade-123` - Go to date with trade selected

### Data Export
- **Journal Entries**: Export your reflections as PDF or text

- **Performance Data**: Export metrics for external analysis
